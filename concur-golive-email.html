Expense Analytics

Interactive expense dashboards — coming soon to Finance & business teams.

Dear Team,

We are pleased to announce that the Concur + Power BI Expense Analytics solution will officially go live.

Starting on Monday, Finance and business teams will have access to a new set of interactive Power BI dashboards connected directly to SAP Concur — giving everyone a faster, richer, and always up-to-date view of expense data, without any change to the existing expense submission process.

What's changing

Starting on Monday, you will be able to use the new Expense Analytics dashboards to:

  • Explore expense data with interactive filters by Cost Center, expense type, employee, and date range — directly in your browser
  • Access automatically refreshed data twice a day, with no manual spreadsheet exports required
  • Run period-over-period comparisons and multi-year trend analysis using the full historical dataset
  • Review Cost Center spend and vendor concentration (for Cost Center owners and Finance leads)
  • Work from a single, shared version of the numbers across Finance and all business units — no reconciliation needed

These dashboards are designed to simplify expense analysis, reduce manual reporting effort, and give every team the context they need to make faster, better-informed decisions.

How to be ready

No software installation is required. The dashboards run entirely in your browser via Power BI Service. If you have not already done so, please make sure you are familiar with the solution before go-live by reviewing the materials below:

Review the following resources on SharePoint:
Dashboard User Guide — overview of all 8 report pages, filters, and navigation
Quick Reference Card — most common use cases and how to apply them
Access & Permissions Guide — how Cost Center views and row-level security work
